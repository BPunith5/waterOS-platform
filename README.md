# WaterOS Platform

Full-stack Smart Water Management Platform — IoT tank monitoring with live telemetry, alerts, and analytics.

- `frontend/` — Vite + React + TypeScript + Tailwind CSS v4 + Framer Motion + React Router + Socket.IO client
- `backend/` — NestJS + Mongoose (MongoDB) + JWT + Socket.IO + scheduled jobs

See `/frontend` and `/backend` for module-specific setup.

See `DEPLOYMENT.md` for deploying to Vercel (frontend), Railway/Render (backend), and MongoDB Atlas.
